Single.js

一个小型的MVC单页应用框架

View on GitHub Download .zip Download .tar.gz

Single.js是什么？

是一个MVC模式的单页应用框架，纯面向对象，所有的组件都是类式继承，在基础层提供事件机制，通过hash的变化来路由来执行View。View提供生命周期事件，来处理你所有的可能情况。 Singlejs提供自带ui组件，提供了最基本的Alert，Toast，Loading，Mask等基本组件，同时你可以通过基类来开发自己的组件，或是扩张现有ui组件。通过Single.js，你能够快速的构建单页应用。

Single.js的由来。

这个框架源至本人在携程时，开发过的单页应用框架Lizard，离开携程后，自己重新开始设计开发一个单页应用框架，摒弃了一些不好的，加入了新的，形成了现有的Single.js。基本上满足工作中的，复杂单页应用的h5项目。
